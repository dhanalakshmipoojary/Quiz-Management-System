*Quiz Management System*

**Assumptions**
- Then application will be build using Nextjs app route
- Mangodb will be used as the database
- Admin will have the authendication using email and password
- For public user authendication not require
- Quiz question can be MCQ,true/false, text

**Scope**
- Admin login and logout
- Admin dashboard which include: create quiz, quiz title, quiz question page
- Public dashboard includes: quiz page , submit answers, see score

**Approach**
- Frontend: NextJS app route
- Backend: NextJS Api
- Database: MongoDB
